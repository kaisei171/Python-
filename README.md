# Python基礎文法クイズアプリ

## 概要

このアプリは、Pythonの基礎文法を学習するための4択クイズアプリです。  
StreamlitとSupabaseを使って作成しました。

Pythonの基本文法100問をクイズ形式で出題し、  
間違えた問題はデータベースに保存され、復習モードで重点的に学習できます。

---

## 主な機能

- Python基礎文法の4択クイズ（100問）
- 解答履歴の保存（Supabaseデータベース）
- 間違えた問題だけを出題する復習モード
- 学習記録の表示
